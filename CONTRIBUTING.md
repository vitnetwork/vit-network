# Contributing to VIT Network

Thank you for helping build Africa's intelligence oracle!

## Setup

Clone the repo, install dependencies with pip and npm, then run `bash scripts/start_fullstack.sh`.

## Workflow

1. Fork and create a branch named `feature/your-feature` or `fix/issue-name`
2. Write clean code with type hints and docstrings
3. Add tests for new functionality — run `pytest tests/ -v`
4. Follow Conventional Commits: `feat:`, `fix:`, `docs:`, `test:`, `chore:`
5. Open a Pull Request against `main`

## Team Areas

| Area | Team |
|------|------|
| Backend / API | Core Engineering |
| AI / ML / Agents | AI & ML Research |
| Smart Contracts | Blockchain & Web3 |
| Frontend / UI | Frontend & Design |
| Infrastructure | DevOps & Infrastructure |

## Code Standards

- **Python**: PEP 8, type hints, async/await
- **TypeScript**: strict mode, functional components
- **Solidity**: NatSpec comments, Foundry tests required
